01

การสมัครสมาชิกและเข้าสู่ระบบ

เริ่มต้นใช้งานระบบสำหรับผู้ใช้ใหม่ และขั้นตอนการกลับเข้ามาใช้งานสำหรับผู้ที่มีบัญชีแล้ว

  1. เมื่อเปิดเว็บครั้งแรก ระบบจะแสดงหน้าล็อกอิน ให้กรอก `ชื่อผู้ใช้งาน` และ `รหัสผ่าน` หากมีบัญชีอยู่แล้ว
  2. หากยังไม่เคยใช้งาน ให้กด `ลงทะเบียนที่นี่` เพื่อเปิดแบบฟอร์มสมัครสมาชิก
  3. กรอกข้อมูลให้ครบ ได้แก่ ชื่อ-นามสกุล, ตำแหน่ง, หน่วยงาน, Username, อีเมล และรหัสผ่าน
  4. กด `ลงทะเบียน` แล้วเข้าสู่ระบบด้วยบัญชีที่สร้างไว้
  5. เมื่อเข้าสู่ระบบสำเร็จ ระบบจะพาเข้าสู่หน้าหลักของผู้ใช้โดยอัตโนมัติ
ข้อแนะนำ

02

เมนูหน้าหลักและภาพรวมของระบบ

ทำความเข้าใจองค์ประกอบหลักของหน้า Dashboard ซึ่งเป็นหน้าที่ใช้งานบ่อยที่สุด

  1. แถบเมนูด้านซ้ายใช้สำหรับสลับไปยัง `หน้าหลัก`, `คู่มือผู้ใช้งาน`, `บันทึกอบรมภายนอก`, `ประวัติการอบรม`, และ `แก้ไขข้อมูลส่วนตัว`
  2. กล่องข้อมูลด้านบนแสดงชื่อผู้ใช้และหน่วยงาน เพื่อยืนยันว่าคุณล็อกอินด้วยบัญชีที่ถูกต้อง
  3. แถบด้านบนของเนื้อหาจะแสดง `ชั่วโมงสะสมรวม` ที่รวบรวมจากการอบรมของคุณ
  4. การ์ดสถิติแสดงจำนวนหลักสูตรที่กำลังเรียน และจำนวนใบประกาศที่ได้รับ
  5. ส่วน `หลักสูตรอบรมทั้งหมด` ใช้ค้นหาและเลือกเรียนหลักสูตรภายในระบบ โดยการ์ดหลักสูตรจะมีป้ายกำกับว่าเป็น `ออนไลน์` หรือ `อบรมในห้อง` รวมถึงป้าย `หลักสูตรบังคับ` เมื่อมีการกำหนดไว้
  6. ส่วน `หลักสูตรภายนอกแนะนำ` ใช้เปิดลิงก์ลงทะเบียนกับหน่วยงานภายนอกโดยตรง

03

การเรียนภายใน 2 รูปแบบและ Flow จนได้รับใบประกาศ

รองรับทั้งหลักสูตรแบบเรียนออนไลน์ผ่านวิดีโอ และหลักสูตรแบบอบรมในห้องที่เข้ามาทำข้อสอบหลังเรียนในระบบ

  1. ที่หน้า Dashboard เลือกหลักสูตรจากบล็อก `หลักสูตรอบรมทั้งหมด` แล้วกดปุ่มตามรูปแบบหลักสูตร เช่น `เข้าสู่บทเรียน` สำหรับคอร์สออนไลน์ หรือ `ทำแบบทดสอบหลังอบรม` สำหรับคอร์สอบรมในห้อง
  2. เมื่อกดเข้าสู่บทเรียน ระบบจะพาเข้าสู่ห้องเรียนทันที ซึ่งประกอบด้วยวิดีโอ เนื้อหารายหน่วย แถบความคืบหน้า และรายการบทเรียนด้านข้าง
  3. ผู้ใช้ต้องเรียนให้ครบทุกหน่วย ความคืบหน้าจะเพิ่มตามการรับชมจริง และระบบจำตำแหน่งที่ดูค้างไว้ให้อัตโนมัติ
  4. ในห้องเรียนมีเมนู `ถาม-ตอบ (Q&A)` สำหรับส่งคำถามเกี่ยวกับเนื้อหาหลักสูตร
  5. เมื่อเรียนครบ 100% ปุ่ม `ทำแบบทดสอบหลังเรียน` จะปรากฏขึ้น ส่วนหลักสูตรอบรมในห้องจะไม่มีวิดีโอและจะพาเข้าสู่ Post-test หลังผู้ใช้ยืนยันว่าเข้าร่วมอบรมแล้ว
  6. หากสอบผ่านเกณฑ์ของหลักสูตร ระบบจะสร้างใบประกาศให้ดาวน์โหลด และบันทึกผลลงประวัติการอบรมโดยอัตโนมัติ หลังจากนั้นจะมีหน้าประเมินความพึงพอใจ
  7. เมื่อทำทุกขั้นตอนครบ ข้อมูลชั่วโมงและใบประกาศจะสะสมในหน้าหลักและประวัติการอบรมของคุณ

04

การลงทะเบียนอบรมภายนอกที่แนะนำ

ใช้สำหรับหลักสูตรที่แอดมินแนะนำจากหน่วยงานภายนอก และเปิดให้ลงทะเบียนผ่าน URL ภายนอก

  1. ที่หน้า Dashboard ให้เลื่อนมาที่บล็อก `หลักสูตรภายนอกแนะนำ`
  2. แต่ละการ์ดจะแสดงชื่อหลักสูตร หน่วยงานที่จัด และจำนวนชั่วโมงที่คาดว่าจะได้รับ
  3. กดปุ่ม `ลงทะเบียน` เพื่อเปิดหน้าเว็บไซต์ของหน่วยงานภายนอกในแท็บใหม่
  4. ระบบนี้ทำหน้าที่เป็นช่องทางแนะนำหลักสูตรและพาไปลงทะเบียน ไม่ได้บันทึกผลการเข้าอบรมภายนอกให้อัตโนมัติ
  5. หลังอบรมภายนอกเสร็จแล้ว ให้กลับมาที่เมนู `บันทึกอบรมภายนอก` เพื่อบันทึกหลักฐานเข้าระบบ

05

ขั้นตอนการบันทึกอบรมภายนอก

สำหรับนำผลการอบรมจากภายนอกเข้ามาเก็บใน Portfolio ของคุณ พร้อมแนบหลักฐาน

  1. เปิดเมนู `บันทึกอบรมภายนอก` จากแถบด้านซ้าย
  2. กรอก `หัวข้อการอบรม`, `หน่วยงานที่จัด`, `วันที่`, และ `ระยะเวลา` เป็นชั่วโมงและนาที
  3. แนบไฟล์หลักฐาน เช่น ใบประกาศ, หนังสือรับรอง หรือรูปภาพที่เกี่ยวข้อง
  4. กด `บันทึกข้อมูลเข้า Portfolio` เพื่อส่งข้อมูลเข้าระบบ
  5. ข้อมูลที่บันทึกแล้วจะถูกส่งเข้าสู่กระบวนการตรวจสอบ/อนุมัติ ก่อนนำไปรวมในประวัติการอบรมอย่างเป็นทางการ

06

เมนูประวัติการอบรม (Portfolio)

ศูนย์รวมประวัติการอบรมทั้งหมด ทั้งจากการเรียนภายในระบบและอบรมภายนอกที่ได้รับอนุมัติแล้ว

  1. เปิดเมนู `ประวัติการอบรม (Portfolio)` เพื่อดูรายการอบรมทั้งหมดของคุณ
  2. ตารางจะแสดงวันที่, ชื่อหลักสูตร/หัวข้ออบรม, หน่วยงานที่จัด, รูปแบบการอบรม, ชั่วโมง และหลักฐาน
  3. ประวัติจากหลักสูตรภายในระบบจะถูกเพิ่มให้อัตโนมัติเมื่อเรียนและผ่านกระบวนการครบถ้วน ไม่ว่าจะเป็นหลักสูตรออนไลน์หรือหลักสูตรอบรมในห้อง
  4. ประวัติจากภายนอกจะปรากฏเมื่อรายการที่ส่งบันทึกได้รับการอนุมัติจากแอดมินแล้ว
  5. กดปุ่ม `บันทึกเป็น PDF` เพื่อสร้างเอกสารสรุป Portfolio สำหรับใช้งานต่อหรือเก็บเป็นหลักฐาน

07

เมนูแก้ไขข้อมูลส่วนตัว

ใช้ปรับปรุงข้อมูลผู้ใช้งานให้เป็นปัจจุบัน เพื่อให้รายงานและใบประกาศของคุณถูกต้อง

  1. เปิดเมนู `แก้ไขข้อมูลส่วนตัว`
  2. คุณสามารถอัปโหลดรูปโปรไฟล์ใหม่ และแก้ไขชื่อ-นามสกุล, ตำแหน่ง, หน่วยงาน และรหัสผ่านใหม่ได้
  3. ช่อง `Username` เป็นแบบอ่านอย่างเดียว ไม่สามารถแก้ไขได้จากหน้านี้
  4. หากไม่ต้องการเปลี่ยนรหัสผ่าน ให้เว้นช่องรหัสผ่านใหม่ว่างไว้
  5. กด `บันทึกข้อมูล` เพื่ออัปเดตข้อมูลในระบบ

08

ฟังก์ชันอื่นๆ ที่ควรรู้

รวมฟังก์ชันประกอบที่ช่วยให้ใช้งานระบบได้สะดวกและครบถ้วนยิ่งขึ้น

ค้นหาหลักสูตร

บนหน้า Dashboard มีช่องค้นหาชื่อหลักสูตร เพื่อช่วยกรองรายการหลักสูตรภายในระบบได้รวดเร็ว

หลักสูตรอบรมในห้อง

บางหลักสูตรไม่มีวิดีโอในระบบ โดยผู้เข้าอบรมจะเข้าร่วมอบรมในห้องเรียนก่อน แล้วจึงเข้ามาทำแบบทดสอบหลังเรียนในระบบเพื่อรับใบประกาศ

ชั่วโมงสะสมรวม

แสดงบนแถบบนของหน้าผู้ใช้ เพื่อสรุปชั่วโมงการอบรมสะสมของคุณจากข้อมูลในระบบ

หลักสูตรบังคับสำหรับสายสนับสนุน

บุคลากรสายสนับสนุนจะไม่ใช้เกณฑ์ 10 วัน/คน/ปี แต่ระบบจะเน้นติดตามการผ่านหลักสูตรบังคับที่ถูกกำหนดไว้ตามกลุ่มเป้าหมายแทน

Q&A ในห้องเรียน

ใช้ถามคำถามระหว่างเรียน โดยกดขยายส่วน `ถาม-ตอบ` ในหน้าห้องเรียน แล้วพิมพ์คำถามส่งได้ทันที

เรียนต่อจากจุดเดิม

ระบบจำตำแหน่งวิดีโอที่ดูค้างไว้และความคืบหน้าของแต่ละหน่วย เพื่อให้กลับมาเรียนต่อได้สะดวก

แบบประเมินหลังเรียน

หลังสอบผ่านและรับใบประกาศ ระบบจะเปิดหน้าประเมินความพึงพอใจ เพื่อใช้พัฒนาหลักสูตรต่อไป

เมนูแอดมิน

เมนู `จัดการระบบ (Admin)` จะแสดงเฉพาะผู้ที่มีสิทธิ์แอดมินเท่านั้น และรายงานจะสามารถกรองแยก `พยาบาลวิชาชีพ` กับ `สายสนับสนุน` ได้เพราะใช้เกณฑ์คนละแบบ

การใช้งานบนมือถือ

บนหน้าจอเล็กสามารถกดปุ่มเมนูมุมบนซ้ายเพื่อเปิดหรือปิดแถบเมนูด้านข้างได้

ออกจากระบบ

หากใช้งานบนเครื่องสาธารณะ ควรกด `ออกจากระบบ` ทุกครั้งเพื่อป้องกันผู้อื่นเข้าถึงข้อมูลของคุณ

บันทึกประวัติการอบรมภายนอก
